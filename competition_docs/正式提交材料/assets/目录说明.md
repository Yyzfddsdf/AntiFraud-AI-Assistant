# 正式提交材料 Assets 目录说明

本目录仅服务于当前比赛正式提交材料，已按文档类型拆分，避免素材混用。

## 目录结构

### `材料二_项目简介PPT`

用于 `提交材料二_项目简介PPT_正式版.md` 及其后续 PPT 制作。

包含：

- `png/`
  - 产品截图
  - 管理员端截图
  - 工程截图
  - 统计类截图
- `svg/`
  - PPT 中使用的结构图、流程图、机制图
- `reference/`
  - PPT 备用说明材料，如动态阈值案例参考

### `材料三_项目详细方案`

用于 `提交材料三_项目详细方案_正式版.md`。

包含：

- `svg/`
  - 详细方案正文中插入的四张正式机制图

## 使用原则

1. 材料二优先引用 `assets/材料二_项目简介PPT/`
2. 材料三优先引用 `assets/材料三_项目详细方案/`
3. 不再使用原 `competition_docs/比赛PPT/assets/` 旧路径
4. 新增图片、视频、音频时，先判断归属材料，再放入对应子目录
